To reboot the system, choose one command among these
	-  reboot
	-  systemctl reboot
	-  shutdown -r now
	-  init 6
	-  telinit 6

To shutdown the system, choose one command among these:

	-  halt
	-  systemctl halt
	-  shutdown -h now
	-  init 0
	-  telinit 0

To switch off the system, choose one command among these:
	 - poweroff
	-  systemctl poweroff
